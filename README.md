# e-com_store
E-Com Store is a fully responsive, frontend-only e-commerce website designed to showcase products with a clean layout and smooth user experience. This project focuses on UI/UX, visual design, and interactive features without any backend or database connection.

 Key Features

Home Page: Clean hero section, featured products, and promotional banners.

Product Listings: Grid view with images, prices, and quick-view options.

Responsive Design: Fully optimized for desktop, tablet, and mobile screens.

Modern UI: Simple navigation bar, footer, and visual components with smooth CSS transitions.

 Tech Stack

HTML5 and CSS3 

 Purpose

This project demonstrates how to build a modern e-commerce interface using only frontend technologies. It can be used for portfolios, UI practice, school projects, or as a starting template for integrating a backend later.

 How to Run

Clone or download the repository

Open the project folder

Run index.html directly in a browser (no server required)
